# Roommate Harmony

Shared living clarity powered by AI. Your apartment runs smoother when everyone's on the same page.

Roommate Harmony is a software assistant that translates household inputs into signals about stability and risk in shared living environments.

## Features
- **AI Mediation Assistant**: Analyzes scheduling conflicts and suggests neutral, fair resolutions.
- **Shared Expense Dashboard**: Simple logging system where roommates record shared purchases.
- **Participation Analytics**: Tracks each roommate's contribution trends over time.
- **Early Warning Alerts**: Flags when a roommate's participation drops significantly.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
